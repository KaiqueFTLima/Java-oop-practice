# ☕ Java OOP Journey — Fundamentos Sólidos em Programação Orientada a Objetos

Este repositório representa minha jornada prática no aprendizado de Programação Orientada a Objetos com Java.

Mais do que exercícios isolados, aqui está a construção da minha base como desenvolvedor back-end, evoluindo desde estruturas simples com vetores até conceitos avançados como herança, polimorfismo e manipulação de datas com a API moderna do Java.

Meu foco é consolidar fundamentos sólidos para atuar profissionalmente na área de desenvolvimento.

---

## 🎯 Objetivo do Repositório

Desenvolver domínio real dos pilares da POO através da prática constante, organização de código e aplicação progressiva dos conceitos.

Cada pasta representa uma etapa da minha evolução técnica.

---
